Using ChatGPT for Web Development

February 8th, 2024


I initially create this website so I could become more familiar with HTML, CSS, and Javascript. In this iteration of my site, I also wanted to see how I could use AI, ChatGPT, in the web development process.

This was my starting point with ChatGPT: Create the HTML and CSS code that showcases my interests for my personal website. I want my website to have a modern and professional look. My interests are Cybersecurity and Apple App Development, my hobbies are dog training and jazz guitar. I would like each one of these areas to have a navigational link to a page that includes more details. Within the header, create a big picture on the right and on the left, my name with ,a two line introduction and a call to action below the header. I also want an about section that introduces me and my bio / interests in about 200 words.

I used ChatGPT to create javascript, stub out my other pages ,and to answer questions as I progressed in developing my site. Overall, the process was very enjoyable and beat trying to use Google to find answers. As I had ideas for my site, I just asked ChatGPT, which produced usable code snippets and suggestions.
